Using the Virtual Fly Brain (VFB) software

The VFB software can be used in several ways:

  • Because the VFB software was developed as part of the VFB project it is open source and freely distributed under a GNU GPL2 license. This means that the software can be embedded in any third-party website where it can provide the same functionality as on the VFB website.
  • Produced image stacks can be uploaded and/or registered with the Viewer and standard brain stacks to deliver third-party results using our Viewer and infrastructure. Please note that this is a work in progress and this functionality is currently unavailable.
