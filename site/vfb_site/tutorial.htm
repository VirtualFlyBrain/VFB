Interactive 3D Viewer overview


An overview of the 3D viewer is given above. Its main components are:

  • (A) Tools for user interaction with the image. These allow the user to specify the content and parameters of a current view and include navigation, zoom, depth and rotation. The zoom tool ('magnification') allows one to choose a resolution level for the displayed tiles. The depth tool ('dst') allows the user to specify which level in the Z-axis (anterior to posterior) the stack is displayed. The rotation tool (p=0 y=0 r=0) provides the means to rotate the stack on the X-, Y- and Z-axis. Using this tool the user has a choice to either define an arbitrary angle to display the stack, or to select one of the 3 preset orientations: frontal, horizontal or sagittal. The navigation tool indicates the currently visible region of the image, and allows easy scrolling of the image by dragging.
  • (B) A 2D section plane through the image stack (Stack Viewer)
  • (C) Autocomplete search box for anatomy terms
  • (D) Interactive anatomy tree
  • (E) Term info panel provides information on the currently selected term
  • (F) Query menu
The viewer is at the heart of the site and appears in different guises throughout (e.g., in the Query Builder). The same main functionalities and principles apply for these interfaces.
