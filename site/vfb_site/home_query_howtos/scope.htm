How search scope works

You can limit the scope of anatomy terms that are searched to 'all neuroanatomy' or 'adult brain'.
Only the terms found in the given scope will be displayed in the autocomplete box, meaning all the subsequent queries will only contain results from the chosen scope.
Please note:

  • The default scope is the adult brain.
  • You can change the scope for each search, from the homepage or the page.
