Files for analysis

Currently on the site (since June 2013), FlyCircuit neurons (Chiang et al., 2011, Curr. Biol. 21(1): 1--11) are identified by their neuron name (eg. fru-M-200266), and not by their gene name (eg. FruMARCM-M002262_seg001) as initially.

The csv file below provides you with the mapping for FlyCircuit identifiers for neurons. It contains the neuron idid (eg. 1), gene name and neuron name. It also contains the VFB ids for each of the neurons, which are used to run all internal queries.

Click here to download.

Ontologies

Our query system runs on the most current version of the Drosophila anatomy ontology, which is developed in collaboration with FlyBase. You can download different versions of the latest release of the anatomy ontology in obo or owl formats from here.

Click here for more information on the difference between the various versions of the ontology. For most users, the version fbbt-simple should be sufficient. Help on how to browse and query the ontology is available here.
